# SQL-RETAIL-SALES-ANALYSIS PROJECT 🚀

This project demonstrates end-to-end SQL skills using a retail sales dataset.

## 📊 Dataset
- Customers
- Stores
- Products
- Sales Transactions

## 🔥 Skills Covered
- Data Cleaning (NVL, COALESCE, NVL2)
- Filtering & Aggregations
- Joins
- Group By & Having
- Case Statements
- Window Functions (RANK, LAG, LEAD)
- NTILE, FIRST_VALUE, LAST_VALUE
- Date Analysis
